# 📊 Credit Card Financial Performance Dashboard

## 🧠 Project Overview

This project is an interactive Power BI dashboard built using credit card transaction and customer data stored in PostgreSQL.  
The objective of this analysis is to generate real-time business insights into customer behavior, spending patterns, and financial performance metrics to support data-driven decision-making.
